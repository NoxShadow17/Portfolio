05 👁 Computer Vision 🧠 AI / ML

Real-Time Hand Detection System

Computer vision system that detects and tracks 21 hand landmarks in real-time using MediaPipe Hands. Built to explore gesture-based interaction — a foundation for AR/VR controls, sign language recognition, or hands-free navigation systems.

How it works: MediaPipe runs pose estimation on each video frame, isolating hand regions and running a lightweight landmark model at ~30fps. Gestures are classified by computing angular relationships between landmark vectors.
Library MediaPipe Hands (Google) + OpenCV for video capture
Challenge Latency under poor lighting — solved with frame preprocessing & confidence thresholding
Extension Mapped gestures to keyboard inputs for hands-free control demo
Python MediaPipe OpenCV NumPy
