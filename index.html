02 🧠 AI / ML

CineMatch — ML Movie Recommender

A hybrid recommendation engine combining SVD-based collaborative filtering with TF-IDF content-based similarity. Powered by the TMDB dataset, with YouTube sentiment analysis layered on top to generate a "Vibe Score" for each movie.

ML Breakdown: Cold-start users go through a genre quiz to seed a preference vector. Returning users get SVD-weighted collaborative recommendations. The system auto-selects which mode to weight based on interaction history depth.
Algorithm SVD matrix factorization + cosine similarity (hybrid)
Data TMDB API → Pandas pipeline → Scikit-learn SVD
Result 3× higher CTR vs random baseline
Python Scikit-learn SVD TMDB API Pandas React

04 👁 Computer Vision 🧠 AI / ML

Real-Time Hand Detection System

Computer vision system that detects and tracks 21 hand landmarks in real-time using MediaPipe Hands. Built to explore gesture-based interaction — a foundation for AR/VR controls, sign language recognition, or hands-free navigation systems.

How it works: MediaPipe runs pose estimation on each video frame, isolating hand regions and running a lightweight landmark model at ~30fps. Gestures are classified by computing angular relationships between landmark vectors.
Library MediaPipe Hands (Google) + OpenCV for video capture
Challenge Latency under poor lighting — solved with frame preprocessing & confidence thresholding
Extension Mapped gestures to keyboard inputs for hands-free control demo
Python MediaPipe OpenCV NumPy

05 🧠 AI / ML

Diabetes Risk Prediction System

A binary classification ML model that predicts diabetes risk from clinical features (glucose, BMI, insulin, age, etc.) using the Pima Indians Diabetes dataset. Includes a clean web interface where users can input values and get instant risk assessment with confidence scores.

Why this problem: Early diabetes detection can significantly reduce long-term complications. This project explores how ML can make clinical screening faster and more accessible — even as a basic educational demo.
Model Logistic Regression + Random Forest (compared via cross-validation)
Preprocessing Missing value imputation, StandardScaler, SMOTE for class imbalance
Accuracy ~82% on test set · AUC-ROC: 0.87
Python Scikit-learn Pandas Matplotlib Flask HTML/CSS

Neurodiverse Support System — Case Study

🔍 The Problem

Students with ADHD and Dyslexia are consistently underserved by productivity and learning tools. Text is too dense, task structures are too rigid, and interfaces cause cognitive overload before any work gets done.

🧠 The Approach

Built a two-pronged solution: adaptive text simplification using LLM APIs (Groq) that rewrites content at a target reading grade level, and micro-task decomposition that breaks any goal into 3–5 manageable sub-steps.

🏗️ Architecture

⚡ Challenges

Defining "simple enough" is subjective. I used Flesch-Kincaid readability scores as a measurable proxy, targeting Grade 4–6 output for maximum accessibility without losing meaning.

📈 Outcome

Users reported significantly lower perceived task difficulty. The accessibility-first design also benefits non-neurodiverse users dealing with information overload — making it broadly useful.

CineMatch — Case Study

🔍 The Problem

Generic recommendation systems push popular content regardless of individual taste. Most student recommenders over-simplify to content-based filters — resulting in obvious "you liked Inception, try Interstellar" suggestions.

🧠 The Approach

Built a hybrid system: content-based filtering using TF-IDF on genres/overviews + collaborative filtering using SVD matrix factorization. The system auto-selects which to weight based on interaction history depth.

Results + Vibe Score

⚡ Challenges

Cold start problem for new users (no history). Solved with a quick onboarding genre quiz to seed the preference vector before collaborative filtering kicks in.

📈 Outcome

Recommendation click-through rate 3× higher vs random baseline. Added YouTube sentiment analysis on trailers to generate a "Vibe Score" — a differentiator no other student project in this space had.
